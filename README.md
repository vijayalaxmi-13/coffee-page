# ☕ Coffee Page Website

A modern and responsive coffee shop landing page built using HTML and CSS. This project focuses on creating a clean user interface with attractive layouts, coffee-themed designs, and a smooth user experience.

## ✨ Features

- Responsive coffee shop website design
- Home, Menu, Hours, and Contact sections
- Attractive coffee-themed UI
- Custom CSS styling and layouts
- Organized HTML structure
- Image integration for better visual experience

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

- index.html - Main landing page
- menu.html - Coffee menu section
- hours.html - Opening hours section
- contact.html - Contact page
- style.css - Main styling file
- style1.css, style2.css, style3.css - Additional styling files
- Images - Website images
